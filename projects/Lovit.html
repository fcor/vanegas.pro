Lovit

Lovit app

The Lovit App is focused in providing condoms and other realated items for a very low cost with a discret delivery pack.

The Challenge

  • Design the purchase interface user flow

  • Define the project foundations

  • Create icons style

  • Create illustrations

To start this project, I had to understand the data that the client provide me, so the first step was to represent the app flow with the information provided by the client.

When I understand the client idea in my head, I start with the most simple step, that is benchmark, with this method you can analyze how other people are boarding similar problems and your mind start to mix your first ideas with the others ideas to concive a great answer to the problem

Foundations

Before starting to design, I always organize the first app's foundations like: colors, typography, hirerchy and app's personality.

Wireframes

This is one of the most funny steps in the process, because it's here when you can create diverse ways to answer the principal problem, with the posibility to fail. You know fail fast and fail cheap :).

Mock-Up

Now with all the products defined with the data that the client gave me, you can start with the approved wireframes. It's always a good practice if you set your foundations ordered in a library.

Icons

Personally I use Adobe Illustrator to create the icons is very important that use a grid system to ensure consistency in the icons size, you should also got in mind used the same width stroke for all the icons. In This case I chose a very simple icons but they represent the products in a direct way.

Prototype

I used After effects to emulate the user behavior with the app. This was the result, hope you like it :)
